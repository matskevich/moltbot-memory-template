# Conventions

## File Naming

### YouTube Audio
Формат: `YYMMDD Имя_Фамилия Ключевое_слово.mp3`

**Принцип:** дата + автор + тема (макс 3 слова).

Примеры:
- `260130 Michael_Levin Memory.mp3`
- `260131 Joscha_Bach Consciousness.mp3`

Критерий: должно быть **сразу понятно про что**, без лишнего шума.
