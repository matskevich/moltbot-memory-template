# YouTube Skill

Автоматическая обработка YouTube URLs — извлечение транскриптов текстом.

## Когда использовать

Когда владелец присылает YouTube URL (youtube.com или youtu.be).

## Что делает

1. Распознаёт YouTube URL
2. Извлекает транскрипт (субтитры или Whisper)
3. Сохраняет по конвенции: `YYMMDD ключевые-слова.txt`
4. Отправляет файл в Telegram

## Именование файлов

Формат: `YYMMDD ключевые-слова.txt`

**Правила:**
- Дата: текущая (YYMMDD)
- Ключевые слова: 2-3 слова из title видео (автор + тема)
- Lowercase, только буквы/цифры/дефисы
- Максимум 40 символов (без даты)

**Примеры:**
- `260131 michael-levin-memory.txt`
- `260131 joscha-bach-consciousness.txt`
- `260131 lex-fridman-ai.txt`

## Как работает

1. Получаю YouTube URL
2. Вызываю `~/clawd/skills/youtube-transcript/yt-transcript-send.sh`
3. Файл автоматом отправляется владельцу

## Технические детали

- Субтитры (если есть): мгновенно, бесплатно
- Whisper fallback: ~$0.006 за час видео (через Groq)
- Cookies для обхода блокировок (уже настроены)

## Пример использования

Владелец шлёт: `https://youtube.com/watch?v=...`

Я:
1. Распознаю YouTube URL
2. Вызываю skill
3. Отправляю транскрипт текстом

**No reply needed** — просто отправить файл.
